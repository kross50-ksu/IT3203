Project Purpose

This site was created for IT3203 to explore the evolution of websites and demonstrate semantic HTML and responsive design. It serves as both a learning artifact and a showcase of technical and creative skills developed throughout the course. Each page reflects iterative improvements in structure, accessibility, and visual polish, aligned with best practices in modern web development.

Tools and Technologies

Built with HTML5 and CSS3, hosted on GitHub Pages, and designed with accessibility and clarity in mind. Development was done using Visual Studio Code, which supported efficient coding and version control through Git commits. Browser inspection tools were used to troubleshoot layout and styling issues, and the Live Server extension in VS Code provided real-time previews during development.
